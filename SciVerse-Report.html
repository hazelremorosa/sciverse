SciVerse

An Interactive Digital Learning Material for High School Science Education

Overview

SciVerse is a comprehensive, interactive web-based learning application designed to support Grade 7-10 Science education in the Philippines. Built as a Progressive Web Application (PWA), it provides offline-capable access to curriculum-aligned content based on the MATATAG Science Curriculum 2023 issued by the Department of Education.

Mission: To make quality science education accessible, engaging, and effective for Filipino students through interactive digital learning materials that bring scientific concepts to life.

Key Features

Interactive Learning Tools

Interactive Periodic Table

Complete 118-element periodic table with detailed information, electron configurations, and category-based color coding.

Virtual Laboratories

Simulated lab experiments including pH Testing Lab and Virtual Microscope for hands-on learning.

Animated Diagrams

CSS-animated visualizations of scientific processes like the Water Cycle, Mitosis, and particle motion.

Interactive Simulations

Force diagrams, Earth-Sun tilt models, and particle motion simulations with play/pause controls.

Assessment & Practice

Lesson Quizzes

Multiple-choice quizzes at the end of each lesson with instant feedback and score tracking.

Flashcard System

Interactive flashcards for key terms and concepts with flip animations for self-review.

Scientific Glossary

Comprehensive glossary of scientific terms with definitions, searchable and organized alphabetically.

Progress Tracking

Track completed lessons and quiz scores with visual progress bars for each quarter.

Gamification System

Achievement Badges

20 unlockable badges across categories: Learning, Mastery, Exploration, and Dedication.

XP & Levels

Earn experience points for activities and level up through 10 ranks from Novice to Science Master.

Study Streaks

Daily streak tracking with calendar visualization to encourage consistent study habits.

Unlockable Themes

Earn special color themes by reaching milestones and completing achievements.

Navigation & Accessibility

Global Search

Search across all lessons and grades with instant suggestions and highlighted results.

Bookmarks

Save favorite lessons for quick access later with persistent bookmark storage.

Font Size Controls

Adjustable text size (small/medium/large) for comfortable reading.

Keyboard Navigation

Arrow key shortcuts for navigating between lessons within a quarter.

Technical Specifications

Technology Stack

HTML5 CSS3 Vanilla JavaScript Progressive Web App Service Worker LocalStorage API

Architecture

  • Single-File Application: Entire app contained in one HTML file (~29,000 lines) for easy deployment and offline access
  • No External Dependencies: Zero runtime dependencies - works without internet after initial load
  • SPA Navigation: Smooth section-based navigation without page reloads
  • Responsive Design: Optimized for tablets and desktop screens with mobile support

PWA Capabilities

  • Installable: Can be installed on devices as a standalone application
  • Offline Support: Full functionality available without internet connection
  • App-like Experience: Standalone display mode with custom theme colors
  • Automatic Updates: Service worker handles cache updates seamlessly

Data Persistence

  • Progress Data: Lesson completion and quiz scores saved locally
  • Gamification Data: XP, levels, badges, and streaks persisted across sessions
  • User Preferences: Font size, bookmarks, and settings remembered

Curriculum Alignment

All content is aligned with the MATATAG Science Curriculum Guide 2023 (Final version for Grades 3-10) published by the Department of Education, Republic of the Philippines. The application covers all required competencies and learning objectives for each grade level and quarter.
